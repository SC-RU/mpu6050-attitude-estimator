# Roadmap

## Completed

- Register-level communication
- Accelerometer calibration
- Gyroscope calibration
- DLPF configuration
- Fixed-rate sampling
- Roll estimation
- Pitch estimation
- Gyroscope integration
- Complementary filtering
- FreeRTOS task-based architecture (STM32)

### Immediate

- Improved attitude initialization
- Real-time roll and pitch visualization
- Data logging

### Near-term

- Higher-rate sampling and performance optimization
- Additional FreeRTOS tasks (e.g. logging, visualization output)

### Long-term

- Quaternion attitude representation
- Madgwick or Mahony sensor fusion
- Kalman filtering
- 3D orientation tracking
- Motor control integration
